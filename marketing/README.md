# 🎬 AUREUS Video Generator

Système pour générer automatiquement des vidéos marketing virales à partir d'une image.

## 🚀 Utilisation

### 1. Installer FFmpeg

**MacOS:**
```bash
brew install ffmpeg
```

**Windows:**
Télécharge depuis https://ffmpeg.org/download.html

**Linux:**
```bash
sudo apt-get install ffmpeg
```

### 2. Placer ton image

Place ton image (ex: `aureus-logo.png`) dans le dossier `marketing/`

### 3. Générer la vidéo

```bash
cd marketing
node video-generator.js aureus-logo.png
```

## 📝 Textes Viraux Inclus

Le système génère 3 variations de vidéos avec des textes optimisés :

1. **Version 1** : Focus sur le tirage quotidien
2. **Version 2** : Focus sur les millions et la viralité
3. **Version 3** : Focus sur les gains et l'urgence

## 🎨 Format

- **Taille** : 1080x1080 (carré, optimal pour X/Twitter)
- **Durée** : 15 secondes
- **Format** : MP4 (H.264)

## ✨ Features

- Textes animés qui apparaissent progressivement
- Effets visuels (zoom, fade)
- Format optimisé pour X/Twitter
- Génération automatique de 3 variations
